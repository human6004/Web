SIMPLE CRUD - EMPLOYEE MANAGER APPLICATION

Folder structure:
- css/style.css
- data/connect-select-db.php
- data/db-schema.sql
- templates/header.php
- templates/footer.php
- index.php
- create.php
- find.php
- find_query.php
- update.php
- update-single.php
- delete.php

How to run with XAMPP:
1. Copy folder simple_crud_employee_manager into htdocs.
2. Open phpMyAdmin.
3. Import file data/db-schema.sql.
4. Make sure MySQL username/password in data/connect-select-db.php matches your machine.
5. Open in browser:
   http://localhost/simple_crud_employee_manager/index.php
